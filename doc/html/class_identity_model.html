Detailed Description

Represents a SQRL identity stored in the S4-format.

IdentityModel holds a vector of IdentityBlock objects, modelling an object-oriented reproduction of the S4 identity storage format. Following this pattern, IdentityBlock holds a vector of IdentityBlockItem objects, each representing a single setting/value.

More information SQRL's storage format can be found at https://www.grc.com/sqrl/SQRL_Cryptography.pdf

See also
IdentityBlock, IdentityBlockItem
