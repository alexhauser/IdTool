Detailed Description

A helper class for connecting GUI representations of IdentityBlockItem instances to their underlying data source.

This helper class is instantiated and populated by the UiBuilder class to establish a connection between the individual GUI representation of block items and their underlying data sources within the IdentityModel. To do so, UiBuilder will turn the ItemConnector instance into a QVariant and attach it to the GUI control using setProperty(), where it can be retrieved within the slot handling the specific user action which needs access to the data within the IdentityModel.
