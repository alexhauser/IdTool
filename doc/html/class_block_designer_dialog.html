Detailed Description

A UI dialog window for designing S4 identity storage blocks.

BlockDesignerDialog offers functionality for visually creating and editing json-templates representing IdentityBlock objects using the GUI.

ItemEditorDialog will be used to edit individual items/settings within the current block.

See also
ItemEditorDialog, IdentityModel, IdentitiyBlock, IdentityBlockItem, IdentityParser

Creates a new BlockDesignerDialog form, using parent as the parent form, and setting the type of the block to be edited/created to blockType.

If a block definition for blockType already exists, m_WorkMode is set to EDIT, enabling editing mode. Otherwise, m_WorkMode is set to ADD, indicating that a new block definition should be created.
