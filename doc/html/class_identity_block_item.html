Detailed Description

Represents a single item/setting within an IdentityBlock.

Note
Values are stored as strings within IdentityBlockItem objects and only converted to their native types once needed.

More information SQRL's storage format can be found at https://www.grc.com/sqrl/SQRL_Cryptography.pdf

See also
IdentityModel, IdentityBlock
