Detailed Description

A class for enabling copy-pasting of identity blocks.

IdentityClipboard holds a single IdentityBlock which can be used for copy-paste operations.

The class adheres to the "singleton" pattern and can therefore not be instantiated directly using the constructor. Instead, call IdentityClipboard::getInstance() to get a pointer to the singleton instance.
