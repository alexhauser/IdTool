Detailed Description

Represents a tab widget visualizing a single identity.

IdentityTab uses a UiBuilder instance to render an IdentityModel on a QScrollArea, which acts as the main "canvas" for drawing the visual identity representation.

It also holds a QFileInfo object, providing information about the source of the identity, which is being used for displaying the tab's title text and tooltip.
